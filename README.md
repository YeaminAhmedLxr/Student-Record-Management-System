# Student-Record-Management-System

A Python-based Student Record Management System developed as part of the Problem Solving and Programming (PSP) course.

Features
- Add Student Records
- View Student Records
- Search Student Records
- Update Student Records
- Delete Student Records

Technologies Used
- Python
- Dictionaries
- Loops
- Conditional Statements

Project Objective
The objective of this project is to efficiently manage student records through a menu-driven Python application.
